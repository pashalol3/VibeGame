# МАСКАРАД — Правила игры и контекст

## Общее описание

Маскарад — это социальная дедукционная игра на 6–10 игроков в браузере (2D).
Игроки перемещаются по особняку с 5 комнатами, в каждой из которых стоит реликвия.
Роли раздаются случайно и скрыты от остальных игроков.

**Краткая цель игры:**
- **Детективы** хотят поймать Вора.
- **Воры** хотят украсть 3 реликвии или дожить до конца 5 раундов нераскрытыми.
- **Дворецкий** помогает Детективам, узнавая роли ночью.
- **Гости и Самозванец** путают карты и помогают своей стороне.

## Роли

| Роль | Количество | Задача |
|------|------------|--------|
| Детектив | 1–2 | Поймать Вора раньше, чем украдены 3 реликвии |
| Вор | 1–2 | Украсть 3 реликвии или остаться нераскрытым 5 раундов |
| Дворецкий | 1 | Ночью узнаёт роль одного игрока и может делиться информацией |
| Самозванец | 1 | Ведёт себя как Гость, но имеет особую карточку |
| Гость | все остальные | Помогать Детективам, обмениваться масками, сбивать Вора |

## Карта

Особняк состоит из 5 комнат. В каждой стоит реликвия:

| Комната | Реликвия |
|---------|----------|
| Зал | Корона |
| Библиотека | Древний фолиант |
| Столовая | Золотая чаша |
| Сад | Изумрудная статуэтка |
| Хранилище | Серебряный ключ |

## Ход одного раунда

Игра идёт максимум 5 раундов. Каждый раунд состоит из 4 фаз.

### ☾ Фаза 1 — Ночь (60 секунд)

Все игроки стоят на месте и не видят действий друг друга.

- **Вор** выбирает одну реликвию, которую попытается украсть в этом раунде.
- **Дворецкий** выбирает одного игрока и узнаёт его текущую роль.

> Как выбрать реликвию/игрока: в игре кликните по комнате (для Вора) или по имени игрока в списке справа.

### 🎭 Фаза 2 — Маскарад (3 минуты)

Все свободно ходят по карте (WASD / стрелки).

**Кража:**
- Вор должен добраться до выбранной реликвии и простоять в её комнате **15 секунд в одиночестве**.
- Если кто-то заходит в комнату — таймер сбрасывается.
- После успешной кражи реликвия считается украденной.

**Обмен масками:**
- Любой игрок может предложить другому обменяться ролями.
- Цель должна ответить: «Согласен» или «Отказываю».
- При согласии игроки меняются текущими ролями и способностями.
- Отказ виден всем.

**Слежка (только Детектив):**
- Детектив один раз за раунд выбирает игрока.
- 30 секунд все видят, в какой комнате находится этот игрок.

### ⚖️ Фаза 3 — Обвинение (90 секунд)

Все обсуждают и голосуют.

- Любой игрок может обвинить другого: «Этот человек Вор».
- Остальные голосуют «За» или «Против».
- Если **больше половины** проголосовали «За» — карточка вскрывается:
  - Вор раскрыт → победа Детективов.
  - Невиновный → обвинитель **теряет голос в следующем раунде**.

**Официальный арест (только Детектив, один раз за игру):**
- Детектив вскрывает карточку игрока без голосования.
- Если угадал Вора → победа Детективов.
- Если ошибся → арестован невиновный, и Воры побеждают.

### 📊 Фаза 4 — Итог (30 секунд)

- Если в этом раунде реликвия украдена — Воры получают +1 к счёту.
- Если никто не арестован — начинается следующий раунд.

## Победа

**Детективы побеждают, если:**
- Вор арестован или разоблачён голосованием.
- Это произошло раньше, чем украдены 3 реликвии.

**Воры побеждают, если:**
- Украдены 3 реликвии.
- Прошли все 5 раундов, а Вор так и не раскрыт.
- Арестован или обвинён невиновный игрок.

## Особые ситуации

- **Обмен масками с Вором:** если Вор согласился поменяться ролью, его новая роль больше не «Вор», но Детективы всё ещё должны поймать настоящего текущего Вора.
- **Дворецкий знает правду:** информация Дворецкого актуальна на момент проверки. После обмена масками роли меняются.
- **Несколько Воров:** у каждого Вора своя цель на раунд; в одном раунде может быть украдено несколько реликвий.

## Управление в игре

| Действие | Как сделать |
|----------|-------------|
| Движение | WASD / стрелки (работает в любой раскладке) |
| Выбрать игрока | Клик по имени в списке справа или клик по аватару на карте |
| Выбрать реликвию (Вор, ночь) | Клик по комнате или кнопка «Выбрать реликвию» |
| Обмен масками | Выбрать игрока → нажать «Предложить обмен масками» |
| Слежка (Детектив) | Выбрать игрока → нажать «Объявить слежку» |
| Обвинение | Выбрать игрока → нажать «Обвинить игрока» |
| Официальный арест | Выбрать игрока → нажать «Официальный арест» |
| Чат | Ввод в нижней панели, Enter |

## Как запустить игру

1. Убедитесь, что установлен Node.js (версия 18 или новее).
2. Откройте терминал в папке проекта `C:/Users/Admin/Desktop/probe/webGame`.
3. Установите зависимости (один раз):
   ```
   npm install
   ```
4. Запустите сервер:
   ```
   npm start
   ```
   Или в режиме разработки (авто-перезапуск при изменениях):
   ```
   npm run dev
   ```
5. Откройте браузер и перейдите по адресу:
   ```
   http://localhost:3000
   ```
6. Введите код комнаты (3–8 латинских букв или цифр) и имя, затем нажмите «Войти в комнату».
7. Повторите шаг 6 в 6–10 разных вкладках/браузерах или устройствах (или попросите друзей).
8. Когда в комнате будет минимум 6 игроков, создатель комнаты (👑) нажимает «Начать игру».

## Технические детали

- Браузерная 2D-игра.
- Мультиплеер через WebSockets (Socket.IO).
- Сервер: Node.js + Express.
- Клиент: HTML5 Canvas + JavaScript.
- Управление: WASD / стрелки.
- Действия через кнопки и клики по игрокам.

## Ограничения текущей версии

- Голосование реализовано как все eligible игроки проголосовали или по истечении времени.
- Голос идёт до половины+1, но в текущей имплементации ждёт всех eligible голосов.
- Официальный арест — только детектив, один раз за игру.
- Для обмена масками нужно быть рядом (60 пикселей).
- Слежка показывает комнату цели всем игрокам 30 секунд.
